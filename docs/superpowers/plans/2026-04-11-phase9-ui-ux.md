# Phase 9: UI/UX改善 実装計画

> **作成日:** 2026-04-11（初版）→ 2026-04-12（デザインファースト方針でリビルド）
> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** アプリの見た目と使い勝手を大幅に向上させる — デザインシステム策定→アセット生成→基盤実装→ホーム画面・カレンダー改善→アニメーション→ビジュアルQA

**Tech Stack:** Next.js 16 (static export), React 19, Tailwind CSS v4, lucide-react, next/font

## 注意事項

- Task 1（DESIGN.md）の結果により、Task 3〜7 の具体的な実装内容（色コード、フォント名、アニメーション詳細等）が確定する。計画内の `DESIGN.md に準拠` と記載された箇所は、DESIGN.md 完成後に具体化すること。
- CSS-only アニメーションを原則とし、Framer Motion 等の追加ライブラリは導入しない（バンドルサイズ抑制）。
- ハイドレーション対策（isHydrated パターン）を Date/Math.random 使用箇所で必ず適用すること。
- 静的エクスポート（`output: "export"`）との互換性を常に確認すること。
