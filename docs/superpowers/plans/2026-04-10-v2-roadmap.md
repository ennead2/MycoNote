# MycoNote v2 ロードマップ

> 作成日: 2026-04-10
> v1.0.0 完了時点での今後の開発計画

---

## 現状 (v1.0.0)

- 図鑑100種（全種プレースホルダー画像）
- 簡易識別（特徴マッチング）/ 詳細識別（Claude Vision）
- 採取記録（GPS・写真・地図）
- 採取計画チャット（Claude API）
- エクスポート/インポート
- PWA オフライン対応
- Vercel デプロイ済み (https://myco-note.vercel.app/)

---

## Phase 7: 図鑑写真導入 ⭐最重要

**Goal:** 全100種に代表写真を入れ、図鑑としての実用性を確立する

### 7a: Wikimedia Commons 写真収集・同梱
- Wikimedia Commons API で CC ライセンス画像を種ごとに取得
- WebP 変換・リサイズ（カード用300px / 詳細用800px）
- `image_local` に同梱パスを設定
- `images_remote` に Wikimedia の追加写真URLを設定
- `source_url` にライセンス元URLを記録

### 7b: Google 画像検索リンク
- 図鑑詳細ページに「Google で画像を検索」ボタンを追加
- クリックで `https://www.google.com/search?tbm=isch&q={和名}+キノコ` を新タブで開く
- オフライン時は非表示 or disabled

### 依存関係
- なし（独立して着手可能）

### 技術メモ
- 写真同梱: 100種×2枚（カード+詳細）×平均30KB = ~6MB
- Vercel の無料枠（100MB）に収まる
- Service Worker でキャッシュ → オフラインで写真閲覧可能

---

## Phase 8: 図鑑拡充（300種）⭐重要

**Goal:** 日本で遭遇頻度の高いキノコを網羅的にカバーする

### 8a: データ追加（200種）
- 食用・毒・不食のバランスを維持
- 特に毒キノコと誤食ペアの網羅を優先
- 全種に traits データ付与（簡易識別対応）

### 8b: 写真収集（追加200種分）
- Phase 7 で構築した収集パイプラインを再利用
- 写真が入手困難な種はプレースホルダー + Google検索リンクで補う

### 8c: 500種への拡充判断
- 300種時点でバンドルサイズ・ビルド時間を計測
- 問題なければ500種に拡充
- 必要なら JSON 分割 or 遅延読み込みを検討

### 依存関係
- Phase 7 の写真収集パイプラインに依存

---

## Phase 9: UI/UX 改善 ⭐重要

**Goal:** アプリとしての見た目と使い勝手を大幅に向上させる

### 9a: ホーム画面の作成
- 現在 `/` は `/zukan` へリダイレクト → 独立したホーム画面を新設
- コンテンツ案:
  - 今月の旬のキノコ（シーズンカレンダーから自動抽出）
  - 最近の採取記録
  - クイックアクセス（図鑑・識別・記録）
  - 安全Tips（ランダム表示）

### 9b: シーズンカレンダー改善
- 各月タブでソート（4月を選ぶと4月が旬のキノコだけ表示）
- 現在月をデフォルト選択
- 横棒グラフの視認性向上

### 9c: デザイン修正・アイコン作成（Gemini活用）
- Gemini でアプリアイコンのデザイン案を複数生成
- カラーパレット・フォントの見直し
- ボトムナビアイコンの刷新
- 毒性バッジのビジュアル改善

### 9d: 動作アニメーション
- ページ遷移アニメーション（Framer Motion or CSS Transitions）
- カードのホバー/タップエフェクト
- フィルター切り替え時のアニメーション
- ローディングスケルトン

### 依存関係
- 9c のデザイン方針が 9a, 9b, 9d の実装に影響
- → 9c を先に方針決定、9a → 9b → 9d の順で実装

---

## Phase 10: コンテンツ充実

**Goal:** 図鑑の情報量を専門書レベルに引き上げる

### 10a: 図鑑説明の詳細化
- description / features フィールドの加筆
- 調理法・保存方法（食用種）
- 中毒事例・応急処置（毒種）

### 10b: 動画・記事リンク
- データスキーマ拡張: `media_links` フィールド追加
  ```typescript
  media_links?: {
    type: 'youtube' | 'article' | 'wikipedia';
    url: string;
    title: string;
    timestamp?: string; // YouTube 用（例: "2:30"）
  }[];
  ```
- 図鑑詳細ページに「関連動画・記事」セクション追加
- YouTube の登場秒数つきリンク（`&t=150s`）

### 依存関係
- Phase 8（種数拡充）と並行可能

```
  ↓

  ↓

  ↓

```

## 技術的制約メモ

| 項目 | 現状 | 300種時 | 500種時 |
|------|------|---------|---------|
| mushrooms.json | ~180KB | ~540KB | ~900KB |
| gzip後 | ~40KB | ~120KB | ~200KB |
| 同梱写真 (WebP) | 0MB | ~18MB | ~30MB |
| 静的ページ数 | 113 | ~313 | ~513 |
| ビルド時間 | ~10s | ~20s(推定) | ~30s(推定) |
| Vercel無料枠 | 100MB | 要確認 | 要分割検討 |
